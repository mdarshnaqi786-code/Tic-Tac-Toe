#  Tic-Tac-Toe Game

An interactive and responsive **Tic-Tac-Toe** game built using **HTML, CSS, and JavaScript**.  
This project demonstrates fundamental web development skills including **DOM manipulation, event handling, game logic implementation, and responsive UI design**.

##  Features

- Two-player gameplay (Player **O** vs Player **X**)  
- **Current Turn Indicator**  
- **Scoreboard** to track wins  
- Automatic **Winner Detection**  
- **Tie Match Detection**  
- **Reset Game** option  
- **New Game** option after each match  
- Attractive winning/tie images  
- Responsive design for different screen sizes  
- Smooth hover animations  

---

##  Technologies Used

- **HTML5** → Structure of the game  
- **CSS3** → Styling and responsiveness  
- **JavaScript (ES6)** → Game logic and interactivity
